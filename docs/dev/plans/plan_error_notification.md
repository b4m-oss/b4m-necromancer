# エラー通知　素案

1. ヘルスチェックが成功した後　250
  - LED緑常点灯
  - 音程アップ系ビープ
2. ヘルスチェック失敗、何らかの失敗。550
  - LED赤常点灯
  - 音程降下系ビープ、数回繰り返す
3. 何らかの動作実行中 251
  - LEDオレンジ、ややゆっくりめ点滅
4. スキャン成功後、書類処理中、アップロード処理中 252
  - LEDオレンジとLED緑、ゆっくり交互点滅
  - ビープなし
5. スキャン成功、しかし書類処理、またはアップロード失敗時 551
  - 赤オレンジ、交互点滅
  - 音程降下系ビープ、1回
6. スキャン成功、アップロード成功 260
  - 音程アップ系ビープ1回
  - LED青、ゆるく明滅10秒
7. スキャン成功、書類処理成功、アップロード部分的に成功ら一部失敗
  - LED青とLEDオレンジ、交互に点滅10秒
